box model

박스모델과 추후에 설명할 Grid는 배워두면 좋지만 어려운 css 개념임

보이는 것 처럼 html의 요소는 위와 같이 content, padding, border, margin으로 이루어져 있음.

이는 F12를 눌러서 위의 개발자 모드를 키면 확인할 수 있음. 개발자 모드에서 위의 사진에 있는 아이콘을 클릭한 뒤,
마우스를 아무 html 요소에 가져가면 해당 요소의 정보가 뜸. 개발자 모드는 유용한 기능이라서 자주 사용하면 좋음
본론으로 들어가서 박스 모델은 위와 같이 content, padding 등으로 이루어진 모델임. 해당 요소를 조정하는 것으로
태그가 차지하는 크기 등을 조절하는 것이 가능함.
또한 p 태그나 h1 태그같이 줄 바꿈이나 다른 태그간에 간격이 생기는 태그를 block level element
a 태그나 strong 태그같이 줄 바꿈이나 다른 태그간에 간격이 생기지 않는 태그를 inline element라고 함.

해당 영상에 박스 모델에 대한 설명이 잘 되어있으니 필요하다면 보는 것을 추천
